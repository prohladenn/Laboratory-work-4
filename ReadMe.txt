Лабораторная работа №4
По предмету "Технология ООП"
По теме "Классы. Наследование. Полиморфизм"

Рассматриваемые вопросы:

1. Раздельное объявление методов класа.
2. Управление динамической памятью.
3. Конструкторы.
4. Наследование.
5. Виртуальные функции.
6. Множественное наследование.

Постановка задачи:

Лабораторная работа представляет собой предложенный исходный код, требующий
доработки со стороны студента. Все задания поставлены непосредственно в исходном
коде, в виде TODO-комментариев.

В работе может потребоваться создавать новые файлы.

В рамках настоящей лабораторной работы требуется закончить описание классов,
при необходимости, используя наследование и добавляя компоненты класса, такие как:

- конструкторы;
- деструкторы;
- поля и методы класса.

Настоящая работа предполагает, что слушателю может потребоваться использовать
множественное наследование и полиморфизм, для выполнения задач, поставленных
в рамках работы.

После завершения описания типов данных, в ркмках настоящей работы, предлагается
создать обобщённый массив фигур, содержащий объекты вссех описаных типов и
описать обобщённый алгоритм поиска фигуры максимальной площади.
